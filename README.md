# template-vue

Scffold out a Vue.js project which uses Poi.

## Install

Using [SAO](https://github.com/egoist/sao).

```bash
# Install sao first
yarn global add sao

# From git repo
sao egoist/template-vue folder-name
# Or from npm
sao vue folder-name
```

[Check out the usage of this template.](/template/README.md)
